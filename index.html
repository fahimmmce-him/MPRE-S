ยังไม่ได้ใส่ค่า Firebase

เปิดไฟล์ index.html แล้ววางออบเจกต์ firebaseConfig ที่ได้จากคอนโซล

  1. Firebase Console > Project settings > General
  2. ส่วน "Your apps" > เพิ่ม Web app (</>) ถ้ายังไม่มี
  3. คัดลอกออบเจกต์ firebaseConfig มาวางแทนของเดิม
  4. เปิดใช้ Authentication (Email/Password) และสร้าง Firestore Database
  5. บันทึกไฟล์แล้ว push ขึ้น GitHub (Vercel จะ deploy เอง)
